# Tagging and Search Documentation

## Overview

This blog supports tagging posts and searching by keyword or tag. Tags help organize posts and make them easy to discover.

## Tagging Posts

- When creating or editing a post, use the Tags field.
- Enter tags separated by commas (for example: django, tutorial, tips).
- New tags are created automatically when they do not already exist.

## Viewing Posts by Tag

- Tags are displayed on the post list and post detail pages.
- Click a tag to see all posts that contain it.
- URL format: /tags/<tag-slug>/

## Searching Posts

- Use the search bar on the post list, tag results, or search results pages.
- Search matches post titles, content, and tags.
- URL format: /search/?q=your+keywords

## How to Test

1. Create or edit a post with tags.
2. Confirm tags show on the post list and post detail pages.
3. Click a tag to verify the filtered results page.
4. Use the search bar to find posts by title, content, or tag name.
